TRADE OFF
This is the quiz event of the committee and it is held in two rounds, this event tests the participants' basic circuit designing skills using limited resources and logical thinking.
SYMPHONY
The participants are required to build a musical fountain that controls both water and light based on the music being played. Skyrocketing footfall is witnessed every year which is exhilarating.
Trailblazer
The participants are required to build an autonomous robot which can traverse a path using the line following technique and accomplish a task specified in the problem statement in the shortest time.
FOXHUNT
The competition starts with a hands on workshop followed by which the participants are required to make a transmitter and a receiver (which tracks down the position of the transmitter). A great place for communication enthusiasts.
Light of the seven
A robotics design competition where the participants need to come up with a bot which can automatically move towards light streak and satisfies the conditions of the problem statements.
Automata
The teams are required to come up with the fastest robot which can complete a real time video feed provided by an over-head camera and image processing to traverse through the arena.
